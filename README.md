# GoSimpleREST

GoSimpleREST is exactly as the title says; a simple REST API written in Go.

## Architecture

GoSimpleREST uses MVC (Model-View-Controller) architecture.

### Project Structure

- README.md
- api/
  - app.go
  - handlers.go
  - responses.go
- go.mod
- main.go
- models/
  - item.go
- store/
  - memstore.go
